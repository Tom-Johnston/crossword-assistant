Crossword Assistant

Use this crossword assistant to solve anagrams and to find words which match a pattern. After being loaded for the first time, the website doesn't need to download anything again and will work offline.

How to use

  1. Add a constraint by clicking on the appropriate button.
  2. Input the letters using ? or . for any unknown letters (the space key will insert a ?).
  3. Add any additional constraints by clicking on the appropriate buttons to add them. A constraint can be removed by using the {% include delete-icon.svg %} button next to the constraint. You can toggle the type of the current constraint by tapping on the type.
  4. Hit the {% include search-icon.svg %} button.
